# Review Notes

This directory stores mandatory finish-review notes for commits that stage `src/**/*.sol`.

Rules:
- Stage at least one `docs/reviews/*.md` file whenever staged files include `src/**/*.sol`.
- Use `docs/reviews/TEMPLATE.md` as the base format.
- Keep findings concrete (file/function + impact).
- Fill the fixed `Impact`, `Docs`, `Tests`, `Verification`, and `Decision` fields with real values.
- Do not leave placeholders such as `TBD`, `<path>`, `<selectors or paths>`, or `yes/no`.
- Include exact verification commands and results from the same working tree state.
